# lit-hunt · 找文献，不再抓瞎

一个**通用型**论文资料查找工具——给所有学科的研究者用，不限于某个领域。

## 它要解决什么

做研究的时候，下面这四件事最容易让人抓瞎：

1. **不知道关键词怎么写**——脑子里的问题，翻译成数据库能搜的词，差着十万八千里
2. **找到一堆文献不知道怎么筛**——搜出来 200 篇，哪 10 篇值得看？
3. **不确定引用格式写对没**——APA7 还好，碰到 MLA / Chicago / Harvard 心里没底
4. **论文大纲不知道怎么下手**——选题有了，但不知道从哪段写起

这个工具就是把这四件事打包起来。

## 当前状态

v0.2（专业版）：
- ✅ **专业学术工作台视觉**：IBM Plex Sans + Fraunces，结构化布局
- ✅ **左侧栏搜索历史**：localStorage 持久化，每次搜索自动入库，可点击回看、可删除
- ✅ **左侧栏收藏**：关键词、文献均可点 ☆ 收藏
- ✅ **多维度筛选 chip**：研究侧重（通用/实证/理论/综述）× 语言（中/英）× 年限
- ✅ **关键词建议（卡片）**：中英双语、含运算符、可复制、可收藏
- ✅ **文献结果（mock 卡片）**：8 篇真实存在的学术文献 mock，按相关度打分排序
- ✅ **引用速查**：APA7 / MLA9 / Chicago / Harvard 四列对照，点击复制
- ⏳ 大纲生成器：占位，等 v0.3
- ⏳ 真实 API：v0.3 接 Crossref / OpenAlex

## 本地预览

不需要任何依赖。直接在浏览器打开 `index.html`，或者：

```bash
cd /Users/chenyu/project/lit-hunt
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
